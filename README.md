# 🚀 Modernizer AI: Enterprise COBOL Transformation Platform

Modernizer AI is a high-fidelity orchestration platform designed to migrate legacy COBOL applications to modern Java 21 / Spring Boot architectures using a 9-layer AI pipeline.

## 🛠 Tech Stack
- **Frontend:** React, TypeScript, Vite
- **Styling:** Tailwind CSS (Premium Slate/Indigo Theme)
- **Animations:** Framer Motion
- **Icons:** Lucide React
- **Navigation:** React Router DOM

## 🏗 System Architecture (9-Layer Pipeline)
1. **Configuration:** LLM selection (Local Ollama vs Cloud API).
2. **Ingestion:** Chunking orchestrator with contextual overlap.
3. **Complexity Scoring:** Resource allocation based on code risk.
4. **Throttling:** Dual-engine TPM/RPM rate limiting.
5. **Self-Healing:** Automatic recovery from API 429s.
6. **Conversion:** Chunk-aware Java/C# generation.
7. **Reconciliation:** Merging partial classes into a unified app.
8. **Refinement:** Agentic AI loop (Compile $\rightarrow$ Test $\rightarrow$ Fix).
9. **Validation:** Final production-ready code verification.

## 🌟 Key Features
- **Guided Onboarding:** Multi-step AI configuration wizard.
- **Human-in-the-Loop:** Manual verification of extracted business rules.
- **Mission Control:** Real-time telemetry, token tracking, and system logs.
- **Code Workspace:** Split-pane comparison between COBOL and Java.
- **Prompt Studio:** Live editing of the AI's system constitution.
